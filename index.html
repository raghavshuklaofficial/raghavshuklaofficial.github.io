Things I've built.

A few projects I'm proud of — from bare-metal OS kernels to production AI systems.

💻 Systems

NexusCore OS

C · x86 Assembly (NASM) · Makefile · QEMU · GRUB

Engineered a bare-metal 32-bit OS kernel from scratch in C and x86 Assembly, booting via a Multiboot-compliant GRUB loader into Ring 0 protected mode on real x86 hardware and QEMU — achieving a sub-second boot time with zero external dependencies.

⚙️ Compilers

Nova Compiler

C · x86-64 Assembly · NASM · GNU Make · Linux ABI

Architected a complete multi-phase compiler in pure C11 translating Nova to native x86-64 NASM assembly targeting the Linux System V ABI — spanning lexical analysis through code generation with an object-oriented modular design across 15,000 lines in 26 modules.

👁 AI / ML

Cortexia

Python · FastAPI · React · PostgreSQL · pgvector · Redis · Docker

Architected a production-grade face recognition platform with a 6-stage trust pipeline: RetinaFace (99.4% detection), ArcFace 512-d embeddings (99.83% LFW accuracy), 4-method anti-spoofing ensemble, pgvector sub-millisecond search on 100K+ embeddings, and Platt-calibrated confidence scoring.

💬 NLP

Nexus Conversational AI

Python · FastAPI · Sentence-BERT · RoBERTa · Docker

Built an enterprise-grade conversational AI engine with a transformer NLU pipeline — Sentence-BERT intent classification, BERT-NER + regex entity extraction, RoBERTa sentiment analysis, and a multi-turn dialogue manager with entity memory — achieving sub-100ms inference latency.
